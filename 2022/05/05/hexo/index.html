GitHub Pages + Hexo 搭建个人博客

安装 Git 和 Node.js

Git: https://git-scm.com/downloads
Node.js: https://nodejs.org/en/

全部默认安装即可。

连接GitHub

打开 Git Bash，设置 GitHub 用户：

创建 SSH 密匙：

一路回车，在 .ssh 目录下会生成密钥。
进入 GitHub 设置，将 id_rsa.pub 的内容添加至 SSH Keys，测试连接：

若提示 Are you sure you want to continue connecting (yes/no/[fingerprint])?
输入 yes， .ssh 目录下会生成 known_hosts
提示 You've successfully authenticated 则连接成功

创建仓库

创建 GitHub 仓库（略），创建本地仓库（空目录），进入本地仓库，初始化并下载必要组件：

修改 _config.yml 的 Deployment 部分：

修改本地博客文件，生成页面并部署：

刚部署完需要等待几分钟才能生效，可以通过 hexo s 预览本地网页。

修改主题

进入 hexo 官网 https://hexo.io/themes/ 选择主题，以 minos 为例，下载主题文件：

修改 _config.yml：

拷贝 themes/minos/_config.yml.example，重命名为 _config.yml 作为主题配置文件。
重新生成页面，若提示缺少组件，则根据提示安装必要组件。安装 hexo-renderer-sass 失败，换淘宝镜像后用 cnpm 重新安装：

编写博客

删除 source/_posts 目录下的 md 示例文件，通过hexo new创建新的 md 文件，编辑并部署即可。
